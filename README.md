## 概要
下記内容をGoogle Chromeで表示させるデジタルサイネージ。（デモ版）  
  
現在日時  
何のゴミの日か  
天気  
電車の遅延情報  
また、0秒の時、何分かを音声でお知らせします。  
  
![ウィンドウ画像](/img/window.png)  

## 構成
index.html  
style.css  
app.js  
media/audio.js...音声ファイル  
img/...デモ用の天気画像（api取得先よりDL）  
config.js...設定ファイル

## API
天気：天気予報 API（livedoor 天気互換）  
https://weather.tsukumijima.net/  
  
電車の遅延情報：公共交通オープンデータセンター  
https://www.odpt.org/  
※利用にはAPIキーが必要です。  
const YOUR_ACCESS_TOKEN = 'KEY';　//config.js 11行目のKEYにAPIを入力  
  
なお、本アプリケーションが利用する公共交通データは、公共交通オープンデータセンターにおいて提供されるものです。  
公共交通事業者により提供されたデータを元にしていますが、必ずしも正確・完全なものとは限りません。  
本アプリケーションの表示内容について、公共交通事業者への直接の問合せは行わないでください。  
  
本デモ版を実行するとAPIの取得は行いませんが、下記変数をfalseにすると設定に基づいてAPIの取得を行います。  
const DEBUG_MODE = true; //config.js 15行目  
※高頻度で取得しないようにお願いします。  
  
## その他
デバッグ時の機能として運行情報の右側の文字をクリックすると表示が切り替わります。  
また、ゴミの日をクリックするとごみ表示が切り替わります。（右端をクリックすると良いです。）  
天気は起動時に取得、表示したら更新しない仕様です。
