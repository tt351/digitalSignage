## 概要
下記内容をGoogle Chromeで表示させるデジタルサイネージ。（デモ版）  
  
現在日時  
何のゴミの日か  
天気  
電車の遅延情報  
また、0秒の時、何分かを音声でお知らせします。  
  
![ウィンドウ画像](/img/window.png)  

## 制作背景
家にはテレビが無く、朝はスマホからradikoでラジオを聞いていました。毎朝スマホアプリを手動で起動する手間があり、自動で起動してくれたら良いなと思い、ラズパイをswitchbotを使って自動で起動し、streamlinkというアプリからradikoを流し、時間が来たら終了するシステムを作りました。  
※ラズパイ(Raspberry Pi):電子工作も可能な小型LinuxPC  
※streamlink：radikoなどのストリーミングサイトをVLCなどのメディアプレイヤーで視聴できるコマンドラインユーティリティ  
※switchbot:コンセントに差して家電の電源の入切をリモートやスケジュール制御できる機器  

また、radikoは少し遅れて配信されるため時報が実際とずれます。そのため、0秒の時、何分かを音声で知らせる機能を付けました。あみたろの声素材工房の素材やずんだもんであらかじめ作成した音声を組み合わせて流すようにしています。  

ここからが本題ですが、ラズパイはディスプレイを付ければ情報も表示できるため、日時や天気、電車の運行情報など欲しい情報を付けていき、デジタルサイネージ化しました。  
このレポジトリは一連のシステムのデジタルサイネージ部分となっています。  

### JavaScript,css,html採用理由  
OSを問わず動作し、参考文献もたくさんあり、汎用性が高いためです。  
開発中はgooglechromeで確認し、本番ではラズパイのブラウザ、chromiumで表示させます。  
今の所ローカル環境にファイルをコピーして運用をしているので、更新の手間があります。ゆくゆくはサーバーに置いておき、読み込みに行くようにしたいと思っています。

## API
### 天気：天気予報 API（livedoor 天気互換）  
https://weather.tsukumijima.net/  
  
### 電車の運行情報：公共交通オープンデータセンター  
https://www.odpt.org/  


なお、本アプリケーションが利用する公共交通データは、公共交通オープンデータセンターにおいて提供されるものです。  
公共交通事業者により提供されたデータを元にしていますが、必ずしも正確・完全なものとは限りません。  
本アプリケーションの表示内容について、公共交通事業者への直接の問合せは行わないでください。  
  
本デモ版を実行してもAPIの取得は行いません。

※）私の利用している路線の遅延情報の取得は公共交通オープンデータチャレンジ限定ライセンス（2025年版）を利用してましたが、2026/3/13を最後にデータの公開が終了されました。（2026年度も公開予定とのことです）  
運行情報が取得できないため、代わりに天気情報を表示するようにしました。  
## デモ版から実際に使うための設定
config.js内の下記変数をtrue->falseにします。設定に基づいてAPIの取得を行います。  
const DEBUG_MODE = false; // 15行目  

天気取得の地域の指定のため、末尾6桁の数値を変更します。  
(なお、数値は右記サイトのidの値です。https://weather.tsukumijima.net/primary_area.xml)  

const CITY_WEATHER_1 = 'https://weather.tsukumijima.net/api/forecast/city/130010';//1行目  
const CITY_WEATHER_2 = 'https://weather.tsukumijima.net/api/forecast/city/130010';//2行目  

それぞれ都市名を入力します。  
const CITY_NAME_1 = '都市名'; //3行目  
const CITY_NAME_2 = 'デモ'; //4行目  
  
今はデータが提供されていないため使えませんが、情報までに。公共交通オープンデータセンターよりAPIキーを取得し、下記KEYを取得したAPIキーに書き換えます  
const YOUR_ACCESS_TOKEN = 'KEY';　//11行目  

鉄道名を入力します  
const TRAIN_NAME = 'デモ東海道線'; //6行目  

下記はJR東日本の東海道線の情報を抽出するために書いてます。必要に応じて書換えをお願いします。  
const API_FILTER = ['', 'odpt:railway=odpt.Railway:JR-East.Tokaido', ''];//8行目  

※APIは高頻度で取得しないようにお願いします。  

ゴミの日の情報は17行目のdustDay関数の内容を書換えます。
